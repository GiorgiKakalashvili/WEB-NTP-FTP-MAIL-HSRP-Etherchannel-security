# WEB NTP FTP MAIL HSRP Etherchannel security

აღნიშნულ ტოპოლოგიაში განხილულია:


1)   Vlan-ების შექმნა და ინტერფეისების რეგისტრაცია;

2)   Switch-ების ინტერფეისებზე ფიზიკური წვდომის დროს უსაფრთხოების დებულებები;

3)   MAC მისამართების ფილტრაციის ხარჯზე დარღვევევბის კლასიფიკაციები;

4)   სამაუწყებლო შტორმისაგან დაცვის ღონისძიებები;

5)   VTP Server და VTP Client რეგულაციები;

6)   DHCP Snooping დებულება;

7)   DHCP სერვისის მიღებისთვის სანდო პორტების ჩამოყალიბება;

8)   END მოწყობილობებისათვის DHCP მოთხოვნათა რაოდენობის განსაზღვრა;

9)   ARP Inspect დებულება;

10)  STP Portfast დებულება;

11)  EtherChannel კონფიგურაცია;

12)  Load Balance კონფიგურაცია;

13)  OSPFv2 კონფიგურაცია;

14)  HSRP პროტოკოლის კონფიგურაცია;

15)  Static Route კონფიგურაცია;

16)  კომპიუტერების TCP/IPv4 ადრესაცია;

17)  DMZ ზონის შექმნა;

18)  WEB Server-ის კონფიგურაცია;

19)  NTP Server კონფიგურაცია;

20)  შუალედურ მოწყობილობებზე NTP Server-ით  დროის გასწორება;

21)  MAIL Server-ის კონფიგურაცია;

22)  FTP TFTP Server-ების კონფიგურაცია;

23)  სერვისებზე წვდომის უზრუნველყოფა.
